Control-arm comparison (TEST vs CONTROL)

The experiment is a 50/50 split on in-scope CIT 1st-attempt Visa+MC credit-rail traffic: half routes to the experiment MID (TEST), the other half continues through the merchant's default cascade (CONTROL). The decision tree above is walked TWICE every week — once for each arm. If both arms degrade together, the cause is upstream of the experiment (merchant- or issuer-portfolio-wide). If only the TEST arm degrades, the new-MID hypothesis is directly supported. This comparison is the key interpretive frame for every pull decision.
